DP汇总

动态规划DP

两个特征:

1.最优子结构

2.无后效性

背包DP

01背包

有n个物品,每个物品价值w_i元,体积v_i元,现在有一个总容量为V的背包,每个物品只有一个,求能装下物品的最大值

状态

dp[i][j] 表示从前i个物品中选出体积为j的背包的最大价值

递推式

不选:那就是在前i-1件里选

选:第i个物品的价值为w[i],体积为v[i],则如果选那就是在前i-1个里体积为j-v[i]的最大价值加上w[i];

dp[i][j] = max(dp[i-1][j],dp[i-1][j-v[i]] + w[i]);

完全背包

有n个物品,每个物品价值w_i元,体积v_i元,现在有一个总容量为V的背包,每个物品只有无数个,求能装下物品的最大值

状态

dp[i][j] 表示从前i个物品中选出体积为j的背包的最大价值

递推式

不选:那就是在前i-1件里选

选:第i个物品的价值为w[i],体积为v[i],则如果选,因为一个物品有无数个那就是在前i个里体积为j-v[i]的最大价值加上w[i];

dp[i][j] = max(dp[i-1][j],dp[i][j-v[i]] + w[i]);
